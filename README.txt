Code s'exécutant sur  Aseba.

Le robot lit deux codes-barres en roulant le long d'une bande dégradée. L'un indique la distance au trou et l'autre l'angle. À la fin e la lecture, le robot s'oriente vers le trou et parcourt la distance qui le sépare du trou.
